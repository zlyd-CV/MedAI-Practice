# Brain Tumor Segmentation (PyTorch)

概述
----
这是一个基于 PyTorch 的脑肿瘤（Brain Tumor）分割项目示例，包含训练、验证、测试（test）和模型管理的基本脚本与说明。使用UNet进行脑肿瘤分割

+ 本仓库代码来源于博客:https://juejin.cn/post/7493716546481831972
+ 运行前请配置好SwanLab环境:swanlab_setting.py里配置你的API
+ 适合刚入门的深度学习初学者
